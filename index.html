🚀 Introducing the Keploy Unit Test Agent (UTG) – Now Live!

We're proud to officially launch the Keploy Unit Testing Agent (UTG) - a game-changer for developers who want to ship faster with confidence.

What is it?
The UTG is a GitHub-integrated agent that automatically generates unit tests for your pull requests, right when you open or update them. It inspects your code diffs, understands what changed, and writes focused, clean test cases - so you don't have to.

Why it's awesome:

  • Instantly generates test cases for PRs
  • Zero setup beyond installing the agent
  • Clean, relevant, and stable tests
  • HIPAA-compliant out-of-the-box
  • Works seamlessly with your CI/CD workflows
  • Boosts PR velocity while maintaining code quality

No more boilerplate. No more last-minute test-writing. The UTG understands your code and delivers only what matters - straight into your GitHub flow.

📊 Say Hello to GitStats – Built for Devs, by Devs

As developers, we love building tools for the community. But one challenge we constantly ran into was tracking the number of downloads per release on GitHub.

While GitHub provides some basic numbers, it lacks a clean and developer-friendly dashboard to help us quickly understand what's going on, especially if you're shipping often.

So we thought, why not just build it ourselves?

That's how GitStats was born - a free, open-source tool built using Go, HTML, and CSS that lets you track and visualize your GitHub repository's downloads and release statistics in seconds.

It's simple, fast, and made for developers who love shipping fast and staying data-aware.
